Инструкции для установления PHP:

- Зайти на сайт: https://dev.azure.com/pocketmine/PHP-Builds/_build
- Скачать нужную версию PHP - 8.0.8
- Извлечь архив с PHP
- В случае с Windows стоит сделать следующие шаги:
 - Распаковать архив Windows
 - Распаковать архив PHP, а это тот, где не написано debug-pack
 - В файлах распакованного архива достать php из папки bin и скопировать сюда.

Остальные случаи еще не предусмотрены.
